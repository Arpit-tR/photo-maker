# PhotoMaker 2.3 — Offline Passport Photo Generator

PhotoMaker 2.3 is a Windows-compatible offline desktop application for generating passport-size photos with precise
dimensions, DPI correction, and print-ready layouts.

Built using **Python**, **Tkinter**, and **Pillow**, the tool is designed for reliability, accuracy, and fast local
processing without any internet dependency.

---

##  Key Features

- Generate **3 × 4 cm passport photos** with accurate real-world sizing
- Automatic **DPI correction** for consistent print output
-  Create **4 × 6 inch (300 DPI)** print-ready photo sheets
-  **Real-time preview** with thumbnail generation
-  Unified preview system:
  - Image thumbnails
  - Metadata inspection
  - Validation checks
-  Improved user workflow efficiency by **~2×**
-  Fully **offline** — no data leaves the device

---

##  Tech Stack

- **Language:** Python 3
- **GUI:** Tkinter
- **Image Processing:** Pillow
- **Platform:** Windows
